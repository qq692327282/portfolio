# 案例 Brief：LUNAE — 纯净护肤品牌

## 客户背景（虚拟）

**品牌名**：LUNAE
**定位**：纯净护肤品牌，主打天然成分 + 极简包装
**目标客户**：25-35岁注重成分安全的都市女性
**价格带**：$38-120
**销售渠道**：独立站 + 小红书 + 线下买手店

## 设计需求

1. **品牌标识**：Logo + 配色 + 字体
2. **产品包装**：精华液瓶 + 面霜罐 + 洁面乳管
3. **社交媒体**：小红书/Instagram 帖子模板（5款）
4. **电商主图**：白底产品图 + 场景图
5. **开箱体验**：包装盒 + 说明书 + 贴纸

## 设计方向

- 色调：米白 + 浅粉 + 低饱和绿
- 字体：细衬线（高级感）+ 无衬线（现代感）
- 风格参考：Aesop 的克制 + Glossier 的年轻感 + La Mer 的高级感
- 包装：磨砂玻璃 + 简洁标签 + 大量留白

## 目标

展示美妆/护肤品类的包装设计 + 社媒内容创作能力。
