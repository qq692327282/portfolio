# 案例 Brief：NŌMA Studio — Shopify 独立站设计

## 客户背景（虚拟）

**品牌名**：NŌMA Studio
**定位**：北欧风极简家居生活品牌，主打陶瓷器皿 + 香薰蜡烛
**目标客户**：25-35岁都市女性，注重生活美学，偏好极简风格
**客单价**：$45-180
**市场**：北美 DTC（Direct to Consumer）

## 设计需求

1. **Shopify 首页设计**：Hero Banner + 产品展示 + 品牌故事
2. **产品详情页**：产品图排版 + 价格 + 描述 + 加购按钮
3. **产品图设计**：白底产品图 + 场景图 mockup
4. **Collection 页面**：分类展示
5. **移动端适配**：响应式设计

## 设计方向

- 色调：暖白 + 浅灰 + 黑色点缀
- 字体：无衬线为主，干净利落
- 布局：大量留白，网格系统
- 风格参考：Aesop 的克制 + HAY 的生活感
- 图片：高质量产品图 + 生活场景 mockup

## 目标

展示 Shopify 电商设计全链路能力：从首页到产品页到结账的完整视觉体验。
